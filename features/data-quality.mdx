# Data Quality

Ensure your data meets quality standards with automated testing and monitoring.

## Quality Dimensions

### Completeness
- Check for null values and missing data
- Validate required fields are populated
- Monitor data freshness and timeliness

### Accuracy
- Validate data against known sources
- Check data format and type consistency
- Verify business rule compliance

### Consistency
- Compare data across different systems
- Check referential integrity
- Validate data transformations

### Uniqueness
- Detect duplicate records
- Validate primary key constraints
- Monitor data deduplication processes

## Test Configuration

```yaml
tests:
  - name: customer_id_not_null
    testType: columnValuesToBeNotNull
    columnName: customer_id
  
  - name: email_format_valid
    testType: columnValuesToMatchRegex
    columnName: email
    regex: "^[\\w\\.-]+@[\\w\\.-]+\\.[a-zA-Z]{2,}$"
  
  - name: order_total_positive
    testType: columnValuesToBeBetween
    columnName: total_amount
    minValue: 0
    maxValue: 1000000
```
