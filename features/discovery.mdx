# Data Discovery

Discover and explore all your data assets in one unified platform.

## Search Capabilities

### Global Search
Search across all data assets using natural language:
- Tables, databases, dashboards
- Columns and their descriptions
- Tags and glossary terms
- Owner and domain information

### Advanced Filters
Filter search results by:
- **Data Source**: MySQL, Snowflake, BigQuery, etc.
- **Asset Type**: Tables, dashboards, pipelines
- **Domain**: Sales, Marketing, Engineering
- **Tags**: PII, Critical, Deprecated

## Asset Exploration

### Database Assets
- **Tables**: Schema, columns, relationships
- **Views**: Underlying queries and dependencies  
- **Stored Procedures**: Parameters and logic

### Dashboard Assets
- **Charts**: Underlying datasets and metrics
- **Reports**: Refresh schedules and usage
- **Data Models**: Relationships and calculations

## Usage Analytics

Track how data assets are being used:
- **Query Frequency**: Most and least used tables
- **User Activity**: Who's accessing what data
- **Popular Assets**: Trending datasets and dashboards
- **Access Patterns**: Peak usage times and patterns

## Collaboration Features

- **Descriptions**: Add business context to data assets
- **Tags**: Organize and categorize your data
- **Reviews**: Rate and review data quality
- **Conversations**: Discuss data assets with your team
